**LLM FRAUD DETECTION SYSTEM**
**Description:**
**This project utilizes Python Programming and specifically, the OpenAi API for the analysis of suspicious emails for potential phishing/scamming. The script sends the contents of an email to the GPT-4o-mini model and the script returns a structured JSON response. In the response, the script returns a scam score out of 100 and an explanation as to why the email is fraudulent or not.**
**Skills Demonstrated:** 
- Python Programming
- API Integration
- JSON Parsing
- Prompt Engineering
